## 配色使用原则

1. **60-30-10 规则**: 主色 60% (背景/大面积)、辅色 30% (卡片/区域)、强调色 10% (CTA/重点)
2. **最多 3 个主色**: 色板中选择 1 主色 + 1-2 辅色，其余为中性色
3. **梯度优先**: 同一色相使用不同明度/饱和度变体，而非加入新色相
4. **避免纯黑纯白**: 永远不使用 `#000` 或 `#FFF`，使用带微弱色调的替代色
5. **品牌一致**: 所有页面的配色 token 保持一致
6. **暗色模式必须测试**: 暗色方案在 OLED 屏和 LCD 屏上表现可能差异很大
7. **深色主色 + 暖色强调 = 安全**: 冷色底 + 暖色点缀的打法天然避免 AI 感
